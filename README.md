## 🧮 Example: Hypotenuse Calculation

Using the **Pythagorean Theorem**:

c = √(a² + b²)

### 🧠 Python Implementation

```python
import math

a = float(input("Side A: "))
b = float(input("Side B: "))

c = math.hypot(a, b)

print(f"Hypotenuse: {c:.2f}")
